OVERVIEW

<라이프 사진전>의 기존의 어둡고 모던한 분위기를 그대로 유지하며
동시에 기능적인 요소를 추가 배치 함으로써 깊고 심플한 느낌의 새로운 <라이프 사진전> 이벤트 페이지를 만들었습니다.
깊고 모던한 분위기를 위해 기존에 쓰던 블랙 컬러를 바탕으로
<라이프 사진전>의 포인트 컬러인 레드를 조금 더 밝은 채도로 바꾸어 홈페이지 가시성을 높였습니다.
사진 전시회 중 가장 역사가 오랜된 전시회로 역사가 유구한 만큼 방문자들에게 풍부한 정보와 역사를 제공하며
방문자들이 쉽게 정보를 찾고 얻을 수 있도록 원프레임 방식으로 정보를 심플하게 배치했습니다.
