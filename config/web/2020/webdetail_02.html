OVERVIEW

구민과의 소통을 강화하고 사용자의 편리를 위한 '소통창구'로서의 역할을 집중적으로 고려하여 디자인하였습니다.
기존의 올드하고 딱딱한 분위기의 홈페이지에서 최신 트랜드를 반영하여 가시성을 높혔습니다.
언제, 어디서든 의회의 정보를 사용자가 확인할수있도록 반응형 홈페이지로 제작되었습니다.

전체적으로 깊고 모던한 푸른색을 사용하여 안정되고 진보적인 분위기를 연출하였으며
상반되는 주황색을 포인트 컬러로 사용하여 신선한 느낌을 더했습니다.
서브페이지에 친숙하고 감성적인 일러스트를 더하여 딱딱한 기존의 의회 이미지를
'구민과 함께하는 도봉구 의회'에 부합하는 친숙한 이미지로 변화시켰습니다.
